# Vim Config
This repo contains my .vimrc and other vim related files

To use:

    cd ~
    ln -s path/to/vim-config/vimrc .vimrc
    git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
    vim +PluginInstall +qall
